# RadioMaster Boxer setup

Steps to change the volume of a RadioMaster Boxer radio:

1. Press the `SYS` button
2. Press `PAGE ->` until you're in RADIO SETUP
3. Scroll down to SOUND (under Batt.Range)
4. Select MODE, and once selected choose:
    1. `QUIET` For no sound at all
    2. `ALARM` For only Alarm notifications like Low battery
    3. `NoKey` For everything except key input
    4. `ALL` For all sound
